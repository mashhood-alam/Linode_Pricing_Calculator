Overview
What the calculator does and what it covers

The Akamai Compute Cost Calculator is a standalone HTML file that lets you browse all Akamai Cloud (formerly Linode) compute plan families, configure quantities, and build a running cost estimate — all without needing an account or internet connection after the file is opened.

Pricing data is sourced from the Linode public API (api.linode.com/v4/linode/types) and the official pricing page at akamai.com/cloud/pricing, as of early 2026.

ℹ️
Prices shown are list prices for core US/EU regions. Distributed compute regions and some specialty regions may have different pricing. Always verify final pricing in the Akamai Cloud Manager before provisioning.

Header
Sticky top bar with the Akamai branding and the Monthly / Hourly billing toggle. The selected mode affects all price displays across the entire page.
Tab bar
Switches between the 6 compute categories. Horizontal scrolling is supported on smaller screens. The active tab is highlighted in Akamai blue.
Plan cards
Responsive grid of plan cards. Each shows specs (RAM, CPU, Storage, Transfer), price, and a quantity selector (defaults to 0). Pressing + immediately adds the plan to your estimate; the count turns blue and the card border highlights. Pressing − back to 0 removes it automatically.
Estimate sidebar
Sticky right panel showing all added items, per-item cost, running total, and action buttons. On screens narrower than 1024 px, the sidebar docks to the bottom of the viewport.
Quick Start
Build your first estimate in under a minute
1
Open the calculator
Open index.html in any modern browser. No internet connection or server is required — all pricing data and logic is self-contained in the file.
2
Choose a plan category
Click a tab — Shared CPU, Dedicated CPU, Premium CPU, High Memory, GPU Compute, or Kubernetes (LKE) — to browse plans in that family.
3
Set quantity — estimate updates instantly
Use the − / + buttons on a plan card to set how many instances you need. The estimate sidebar updates in real time with every press — no "Add to Estimate" button required. The count turns blue and the card gets a blue border while it is in your estimate. Decrement back to 0 to remove it.
4
Repeat across categories
Switch to other tabs and add more plan types. The estimate accumulates across all categories — you can mix Shared, Dedicated, GPU, and LKE clusters in one estimate.
5
Copy or clear
Click 📋 Copy Estimate to copy a formatted plain-text summary to your clipboard for sharing or pasting into a doc. Click Clear to reset and start over.
💡
Use the Monthly / Hourly toggle in the header to see prices in the unit that makes most sense for your planning. Both values are always shown in the secondary line on each card.

Shared CPU   10 plans · Starting at $5/mo

⚠️
Sustained CPU usage above ~80% is not recommended on Shared plans. For workloads that need consistent, full-core performance, use Dedicated or Premium CPU instead.

Kubernetes (LKE)
LKE   Interactive cluster builder with 3 control plane tiers

The LKE tab provides an interactive cluster cost builder rather than a fixed plan grid. You configure two independent dimensions — the control plane tier and the worker node pools — and the calculator shows the combined monthly cost.

Control Plane Tiers

Standard
Free
Shared control plane managed by Akamai. Suitable for dev, test, and non-critical production clusters.
High Availability
+$60/mo
Redundant control plane across multiple availability zones. Recommended for production clusters with uptime SLAs.
LKE Enterprise
+$300/mo
Dedicated control plane + all enterprise features including enhanced support. For business-critical and regulated workloads.

Node Pools

Each node pool consists of a node type (Shared, Dedicated, or High Memory) and a node count. The pool cost is simply node_price × node_count. You can add multiple pools of different types to model heterogeneous clusters.

ℹ️
LKE pricing formula:
Cluster cost = Control plane fee + Σ(node_monthly_price × node_count)

For example: HA control plane ($60) + 3× Linode 4GB nodes ($24 each) = $132/mo

Click + Add to Estimate in the LKE cluster summary bar to add the entire configured cluster as a single line item in your estimate sidebar. You can add multiple clusters with different configurations by adjusting the control plane tier and node pools between each click.

Once added, clusters appear in an ✅ In Your Estimate block at the top of the LKE tab — each entry shows the cluster name, its node pool summary, the monthly cost, and a ✕ Remove button. You can also remove a cluster via the ✕ in the estimate sidebar.

Available Node Types in LKE

Node pools support Shared CPU, Dedicated CPU, and High Memory instance types. GPU nodes are not available in LKE node pools via this calculator. Available node options span from the Nanode 1GB ($5/mo) up to the Linode 192GB ($1,152/mo) on the Shared side, and Dedicated 256GB ($2,304/mo) on the Dedicated side.

Billing Model
How Akamai Cloud calculates your actual invoice

Akamai uses a hybrid hourly billing model with a monthly cap. You are charged per hour for what you use, but once your accumulated hourly charges reach the plan's monthly cap, you are not charged further for that resource that month.

Billing unit
Hourly, rounded up to the nearest full hour.
Monthly cap
Usage never exceeds the stated monthly price. Once the cap is reached, that resource is effectively "free" for the rest of the month.
Inbound transfer
Always free. Does not count against your transfer quota.
Outbound transfer
Each plan includes a monthly outbound transfer pool. Transfer quotas are pooled across all instances on the account.
Transfer overages
$0.005 per GB over the pooled quota. This is shown as a reminder note in the estimate sidebar.
LKE control plane
Billed separately from worker nodes. Standard tier is free; HA adds $60/mo; Enterprise adds $300/mo per cluster.
💡
The calculator's Hourly toggle lets you see what a plan costs per hour. These are the rates that appear on your invoice. Multiply by 744 (avg hours/month) to get the effective monthly cost for a full month of use.
Estimate & Export
Managing items in the sidebar and copying your estimate

Adding items

Press + on any plan card — the estimate sidebar updates immediately. There is no separate "Add to Estimate" button. The quantity counter turns blue and the card gains a blue border while it is active in your estimate. Every subsequent + or − press adjusts both the on-card counter and the sidebar total in real time.

Removing items

There are three ways to remove a plan from the estimate:

  • Press − on the plan card until the quantity reaches 0 — it is removed automatically.
  • Click the ✕ button next to any line item in the estimate sidebar — this zeroes and removes it instantly.
  • For LKE clusters, use the ✕ button in the In Your Estimate block at the top of the Kubernetes tab, or the ✕ in the sidebar.

Clearing all items

The Clear button in the sidebar footer removes all items from the estimate at once.

Copying the estimate

Pricing Notes & Disclaimers
What to be aware of before using these estimates
⚠️
Estimates only. Prices in this calculator reflect Akamai's published list prices as of early 2026 for standard US/EU regions. Always confirm current pricing in the Akamai Cloud Manager or at akamai.com/cloud/pricing before committing to a deployment.
What the calculator does NOT include
  • Block Storage volumes — priced separately ($0.10/GB/mo)
  • NodeBalancers — load balancers billed at $10/mo each
  • Object Storage — S3-compatible storage with its own pricing tier
  • Managed Database add-on — additional fee on top of the underlying instance
  • DNS Manager, Backups, Cloud Firewall — optional add-on services
  • Distributed compute region pricing — may differ from core region prices
  • Support plans — Professional or Enterprise support tiers billed separately
GPU plan availability
GPU plans — especially Quadro RTX 6000 and RTX PRO 6000 Blackwell — are marked Limited and may not be available in all regions. Contact Akamai sales or check the Cloud Manager for current availability.
LKE hourly approximation
The hourly equivalent shown for LKE clusters is calculated as monthly_cost ÷ (24 × 30) — a rough approximation. Actual billing is based on per-resource hourly rates for each node individually.
